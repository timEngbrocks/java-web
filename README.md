# java-web

A runtime environment for Java based on TypeScript.

## Currently implemented instruction categories

- Constants
- Loads
- Store
- Stack
- Math

## Missing instruction categories

- Conversions
- Comparisons
- References
- Control
- Extended
- Reserved
